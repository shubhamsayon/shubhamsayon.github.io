INTRODUCTION TO DOCKER

Docker is a container management service.

The concept of docker revolves around the idea that developers can easily develop applications, ship them into containers which can be deployed anywhere.

Containers

A container is a standard unit of software that packages up code and all its dependencies so the application runs quickly and reliably from one computing environment to another. In other words containers are an abstraction at the application layer that packages codes and dependencies together. This means instead of simply shipping the application , containers ship the application as well as the runtime environment needed to run the application.

Containers vs VMs

Virtual machines and containers differ in several ways, but the primary difference is that containers provide a way to virtualize an OS so that multiple workloads can run on a single OS instance. With VMs, the hardware is being virtualized to run multiple OS instances. Containers’ speed, agility, and portability make them yet another tool to help streamline software development.
