CONTAINER COMMANDS

$ docker ps to see the running containers
$ docker ps -a to see all existing containers
$ docker run container_id to run a container
$ docker stop container_id to stop a container
$ docker run -d container_id to run container in background 
$ docker exec -it container_id  /bin/bash to run container in interactive mode
$ docker rm contaier_id to remove container
$ docker rename container_oldName container_Newname to rename a container
$ docker rm container1_id container2_id container3_id delete multiple containers at once
$ docker rm container_name --force  forcefully delete a running container

DOCKER IMAGE COMMANDS

$ docker images to list the docker images                          
$ docker images image_name to list diff versions of a particular image
$ docker image pull image_name:imsge_version to pull a specific version of an image
$ docker image pull --all-tags nginx to download all versions of an image
$ docker image rmi image_name to remove an image

To Push an Image to Docker Hub

$ docker login (enter username and password)

$ docker tag image_name repo_name/image_name:v1 (here image_name:v1 is just the tag for ur image on docker hub)

$ docker push repo_name/image_name:v1 

Port Forwarding/Mapping

$ docker run -itd -p 8001:80 container_id
$ docker port container_name  (this will display the port mapping)
