DOCKER ARCHITECTURE

Note:

  • Docker file is BUILD
  • Docker Image is SHIPPED
  • Docker Container is RUN 

Docker has 3 main components:

  1. Docker Client
  2. Docker Host 
  3. Docker Registry

Docker Client

Machine or medium through which users interact with docker host. The two basic ways of interacting are Docker CLI and Docker API's.

Docker Host

This is the machine which actually performs the task of containarization. It runs the docker daemon which listens to and performs the actions asked by docker client. It builds the docker file and turns it to docker image.

Docker Host

It is the placeholder for docker images and makes the images available to other users.
