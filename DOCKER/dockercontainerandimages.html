DOCKER CONTAINER AND IMAGES

Docker Image

 A Docker image is a file, comprised of multiple layers, used to execute code in a Docker container.

Thus docker image is-

  • Stack of multiple layers created from dockerfile
  • Recognized by Image name or Image ID
  • Can be pushed to or pulled from DockerHub

Docker Hub

This is the Docker registery where you can download images build by other communities as well as push your own images.

Official site for Docker hub − https://www.docker.com/community-edition#/add_ons
