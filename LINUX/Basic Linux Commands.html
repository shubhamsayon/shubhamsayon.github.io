BASIC Linux COMMANDS

cat command : Creating and displaying text files

cat>filename To create a file
cat filename To view contents of file
cat>>filename To edit or append an existing file
cat file1_name file2_name >> file3_name to merge data of file1 and file2 into file3

touch command: To create a file of 0 Byte size

touch filename To create a zero byte file
touch file1 file2 file3 to create multiple zero byte files
touch filename/directoryname to change time stamp of file or directory

NOTE: 

By using CAT command, we can easily create a new file or
view the file.
Touch command is mainly used for creating a empty file. We
cannot insert any content into this file. File storage is 0
bytes of memory.

ls command: listing files and directories

ls  list files in the directory
ls -l long listing along with attributes 
ls -a  all files present in directory including hidden files
ls -F to vie thw type of file inside the directory. i.e. if name of file ends with @ then it is a link, if it ends with * then it's an executable file, if it ends with / then it is a folder
ls -R list files recursively (displays the nested files and folders)
ls -d directoryname list contents of a particular directory

tree command: Used to display files and folders in a user readable tree structure.

tree display files and folders within folder in a tree structure
tree -d list only directories in tree structure
