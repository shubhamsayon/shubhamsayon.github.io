PERMISSIONS AND SYMBOLS 

SYMBOL PERMISSION
r Read
w Write
x Execute
- None

PERMISSION FILE VS DIRECTORIES 

PERMISSION FILE DIRECTORY
r Allows a file to be read. Allows file names in the directory to be read.
w Allows a file to modified. Allows entries to be modified within the directory.
x Allows the execution of a file. Allows access to contents and Metadata for entries.

 

PERMISSION CATEGORIES

SYMBOL PERMISSION
u User
g Group
o Other
a All

Command to change permission: chmod

Numeric Based Permissions

Octal Binary String Description
0 0 --- No Permissions
1 1 --x Execute Only
2 10 -w- Write Only
3 11 -wx Write and Execute
4 100 r-- Read Only
5 101 r-x Read and Execute
6 110 rw- Read and Write
7 111 rwx Read, Write and Execute
