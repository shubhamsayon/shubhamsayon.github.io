SHELL & KERNEL

Kernel

Kernel is the core or the heart of the Operating System.

It is the layer/bridge between the Software and the Hardware layer.

When an OS loads, the Kernel gets loaded first and reains in the memory until the OS is shutdown again.

Kernel is responsible for :

  • DISK MANAGEMENT
  • TASK MANAGEMENT
  • MEMORY MANAGEMENT
  • FILE MANAGEMENT
  • PROCESS MANAGEMENT

 

Shell

Shell is the default interface to Linux that allows the user to interact/access OS services. It accepts commands from the user and executes those commands.

Eg: C Shell, Bourne Again Shell etc.
