Working with Directories

Directories are folders or containers that contain other fiiles or directories

.          represents present directory

..         represents parent directory

Navigating Directories

cd directory_path To change directory
cd .. Change directory one level back
cd ../.. Change directory two level back

Creating and Removing Directories

mkdir directoryname create a directory
mkdir -p dir1 dir2 dir3 create multiple directories
rmdir -p dir1 dir2 dir3 remove empty directories 
rmdir -rf directoryname remove non empty directories
